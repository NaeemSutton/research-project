Setting Up Our Code

Welcome to the setup instructions for our Hilbert Curve for ROS 2 Robots code repository.

Overview

This guide provides step-by-step instructions on how to download and use the code from our repository.

Downloading the Code

  1. Open a terminal.

  2. Clone the repository using the git clone command:

    git clone https://github.com/NaeemSutton/Hilbert-Curve-for-ROS-2-Robots.git

  3. Navigate to the downloaded directory:

    cd .../Hilbert-Curve-for-ROS-2-Robots

Downloading the Code

  1. Ensure that the Python script has execute permissions. If not, use the chmod command to add execute permissions:

    chmod +x hilbert_curve_navigation.py
  2. Make sure ROS 2 is properly installed on your system. If not, you can follow the instructions provided earlier.

  3. Source your ROS 2 installation:

    source /opt/ros/humble/setup.bash
  4. Build the workspace:
    colcon build
  5. Source your workspace:
    source install/setup.bash
  6. Run the Python script using ROS 2 commands:
python3 nav.py
